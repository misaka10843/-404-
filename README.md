# 一个随便写的404页面
一个随便写的404页面，灵感来源于腾讯云域名没备案的那个提示页面
<hr>

## 成品预览
<br>
![预览图](https://i.loli.net/2021/04/06/aWGuYFBDmj7KeOw.png)
<hr>

## 文件内容
<br>
文件分级如下

```
  |-404.html
  |-404
     |--404.js
     |--404.css
     |--404.svg
     |--global.css
```

其中，您可以在404.html中修改您的相关信息,比如您的twitter链接，邮箱，网站链接，以及网站名称
404.css和global.css中您可以修改网站的相关配色（如果更改了还请也更改404.svg颜色以保证颜色不冲突）
